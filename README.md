# Wedding Flow Pro

A comprehensive wedding planning platform.

## Features

- Venue Directory with filtering
- Vendor Management
- Real Weddings Showcase
- Admin Dashboard
- User Authentication

## Getting Started

To run the project locally:

```sh
npm install
npm run dev
```

## Technologies

- React
- TypeScript
- Vite
- Supabase
- Tailwind CSS
- Shadcn UI
